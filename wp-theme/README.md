# WordPress тема "Sculptura"

Тема WordPress для сайта студии красоты Sculptura с поддержкой услуг, акций и отзывов.

## Требования

-   WordPress 5.0+
-   PHP 7.4+
-   Плагин **Advanced Custom Fields (ACF)** — обязательно (для редактируемых блоков)

## Установка

1. **Скопируйте папку темы:**

    - Скопируйте папку `wp-theme/` в `wp-content/themes/sculptura/` вашего WordPress сайта

2. **Установите плагин ACF:**

    - В админке WordPress: Плагины → Добавить новый
    - Найдите "Advanced Custom Fields" и установите/активируйте

3. **Активируйте тему:**

    - Внешний вид → Темы → найдите "Sculptura" и активируйте

4. **Скопируйте ассеты (CSS/JS/изображения):**
    - Соберите фронтенд через Gulp из корня репозитория
    - Скопируйте собранные файлы:
        - `css/style.css` → `wp-content/themes/sculptura/assets/css/style.css`
        - `js/bundle.js` → `wp-content/themes/sculptura/assets/js/bundle.js`
        - `images/*` → `wp-content/themes/sculptura/assets/images/`

要好 (или используйте исходные файлы из `src/images/`)

5. **Настройте главную страницу:**

    - Настройки → Чтение → выберите статическую страницу для главной (создайте страницу "Главная")
    - В админке откройте эту страницу и заполните ACF поля:
        - Hero: заголовок, подзаголовок, изображения (галерея)
        - Features: элементы (репитер с иконкой, заголовком, текстом)
        - Reviews: отзывы (репитер с именем, текстом, фото, датой)

6. **Настройте меню:**
    - Внешний вид → Меню → создайте меню и назначьте его в "Основное меню"
    - Для логотипа в центре меню добавьте пункт меню с CSS-классом `logo` (или в названии укажите "logo")

## Как редактировать контент

### Главная страница

-   Откройте главную страницу в редакторе
-   Заполните ACF поля:
    -   **Hero**: заголовок, подзаголовок, галерея изображений
    -   **Features**: добавьте элементы преимуществ (иконка, текст)
    -   **Reviews**: добавьте отзывы клиентов

### Услуги

-   Услуги → Добавить новую
-   Заполните:
    -   Заголовок
    -   Краткое описание (excerpt)
    -   Основной контент (редактор)
    -   Изображение записи (миниатюра — для карточки на главной)
    -   ACF поле "Изображение вверху страницы" (для детальной страницы)
    -   ACF поле "Интро-текст" (для детальной страницы)

### Акции

-   Акции → Добавить новую
-   Заполните заголовок, текст, изображение записи (миниатюра)

## Структура темы

-   `style.css` — заголовок темы
-   `functions.php` — функции темы, регистрация CPT, ACF поля
-   `header.php` — шапка сайта
-   `footer.php` — подвал сайта
-   `index.php` — шаблон главной страницы
-   `single-service.php` — шаблон детальной страницы услуги
-   `single-sale.php` — шаблон детальной страницы акции
-   `page.php` — шаблон обычных страниц
-   `class-sculptura-walker-nav-menu.php` — кастомный Walker для меню
-   `assets/` — CSS, JS, изображения (скопируйте из сборки)

## Зависимости

-   Splide.js (слайдер) — подключается через CDN
-   jQuery — встроен в WordPress
-   ACF — обязательно для работы темы

## Примечания

-   Прайс-лист разрабатывается отдельно (не включен в тему)
-   Форма записи на прием (секция #reception) — добавится позже с интеграцией Telegram
-   Все изображения должны быть оптимизированы перед загрузкой
