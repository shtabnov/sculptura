# Настройка Telegram для формы записи

Форма записи на прием отправляет уведомления в Telegram вместо email.

## Как получить токен бота

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и создайте бота
4. Скопируйте токен бота (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Как получить Chat ID

### Вариант 1: ID личного чата
1. Напишите вашему боту любое сообщение (например, "Привет")
2. Откройте в браузере: `https://api.telegram.org/botВАШ_ТОКЕН/getUpdates`
3. Найдите в ответе `"chat":{"id":123456789}`
4. Скопируйте это число (это ваш Chat ID)

### Вариант 2: ID группы/канала
1. Добавьте бота в группу или канал
2. Дайте боту права администратора (для каналов)
3. Отправьте сообщение в группу/канал
4. Откройте: `https://api.telegram.org/botВАШ_ТОКЕН/getUpdates`
5. Найдите `"chat":{"id":-123456789}` (для групп ID отрицательный)

## Настройка в WordPress

### Способ 1: Через wp-config.php (рекомендуется)

Добавьте в файл `wp-config.php` перед строкой `/* That's all, stop editing! */`:

```php
// Настройки Telegram для формы записи
define('TELEGRAM_BOT_TOKEN', 'ваш_токен_бота');
define('TELEGRAM_CHAT_ID', 'ваш_chat_id');
```

### Способ 2: Через опции WordPress (временное решение)

Выполните в функции темы или через плагин:

```php
update_option('sculptura_telegram_bot_token', 'ваш_токен_бота');
update_option('sculptura_telegram_chat_id', 'ваш_chat_id');
```

## Проверка работы

1. Заполните форму записи на сайте
2. Проверьте Telegram - должно прийти сообщение с данными заявки
3. Если сообщение не пришло, проверьте логи WordPress (обычно в `wp-content/debug.log`)

## Безопасность

⚠️ **Важно:** Токен бота и Chat ID — это конфиденциальная информация. Не публикуйте их в открытом доступе.

Рекомендуется хранить их в `wp-config.php`, который не должен быть доступен через веб-сервер.
